
# SHERLOCK SCHOLES

Telegram Mini App football Alias game.

Core idea:
Players explain football players instead of regular Alias words.

Main pillars:
- Fast rounds
- Social gameplay
- Telegram-native UX
- Realtime multiplayer
- Football culture

MVP:
- Create room
- Join room
- Team setup
- Random football cards
- Timer
- Score system
- Winner logic

Recommended Stack:
Frontend:
- React
- Vite
- Tailwind
- Telegram SDK

Backend:
- Supabase
- PostgreSQL
- Realtime sync

Future Features:
- Ranked mode
- AI narrator
- Voice recognition
- Club packs
- Seasonal events
