# Шерлок Скоулс — контекст проекта

## Что это
Telegram Mini App, игра Alias про футбол. Карточки игры в Supabase
(таблица cards). football_scraper — офлайн-инструмент сбора игроков,
запускается вручную, в Mini App не входит.

## Стек/ресурсы
Supabase проект konoavrduynecxblqfvq. API-Football free (100 req/день,
сезоны только 2022-2024). Wikimedia pageviews (2000/день). Ключи в .env.

## Таблицы
- players_meta: канон игроки (api_football_id UNIQUE, name_en, name_ru).
- player_seasons: игрок×лига×сезон + pageviews. season=2023.
- cards: игровые карточки (~2688), есть поле pageviews.

## РЕШЕНИЯ (зафиксированы)
- Только сезон 2023, без динамики.
- Порог попадания в игру: pageviews > 3000.
- Сложность (тумблер на главной): лёгкая >19000, сложная >3000.
- Клубы: топ-4 + средние клубы со звёздами (списки в config teams_filter).
- В cards добавлять только тех, кого ещё нет (фаззи-дедуп canonical_key).
- name_en писать в cards при переносе (для EN-переключения в игре).
